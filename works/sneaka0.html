SNEAKA0 is the product of a generative design project, in this case with the purpose of recreating a program from the late seventees, which made use of the routine SNEKAO (Schneeberger Karos).

Afterwards, the project was to be made more complex, adding to what had been initially created.

In adition to the changes in aesthetic, a dynamic aspect was added: beyond the initial function which would only run once (at the beginnning), a recursive algorithm was added.

Starting at a random point, it then selects a random neighbor and flips its orientation (from block to lines and vice-versa), all the while marking its trajectory through an incrementation in color for each block it passes through.
